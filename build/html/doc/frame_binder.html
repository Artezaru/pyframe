from_frame_to_frame(input_name: str | None = None, output_name: str | None = None, *, point: ndarray | None = None, vector: ndarray | None = None) → ndarray | None[source]#

Convert a point or vector from one frame to another frame.

if point is not None and vector is not None, the function will raise a ValueError. if point is None and vector is None, the function will return None.

if name is None, the function will return the point or vector in world coordinates.

Args:

input_name (str, optional): The name of the input frame. Defaults to None. output_name (str, optional): The name of the output frame. Defaults to None. point (np.ndarray, optional): Point in input frame coordinates with shape (3, N). Defaults to None. vector (np.ndarray, optional): Vector in input frame coordinates with shape (3, N). Defaults to None.

Returns:

np.ndarray: Point or vector in output frame coordinates with shape (3, N).

get_compose_frame(input_name: str | None = None, output_name: str | None = None) → Frame[source]#

The return frame is the transformation between the input frame and the output frame.

if input_name is None, the function will return the worldcompose frame of the output frame. if output_name is None, the function will return the inverse worldcompose frame of the input frame.

Args:

input_name (str, optional): The name of the input frame. Defaults to None. output_name (str, optional): The name of the output frame. Defaults to None.

Returns:

Frame: The composed frame with the given name.

Raises:

ValueError: If the frame name does not exist.
